Aimz Documentation

To use Aimz, load the script using:

local aimz = loadstring(game:HttpGet('https://pastebin.com/raw/pzhQD7PD'))()

Functions

Enable or Disable Aimbot:

aimz.aimbot(true) -- Enable
aimz.aimbot(false) -- Disable

Set Aim Part:

aimz.aimpart("Head") -- Aim at the head
aimz.aimpart("HumanoidRootPart") -- Aim at the body

Set Aim Prediction:

aimz.prediction.x(0.2) -- X axis prediction (0-1)
aimz.prediction.y(0.1) -- Y axis prediction (0-1)

Set Camera Shake:

aimz.shake.x(0.1) -- X axis shake (0-1)
aimz.shake.y(0.1) -- Y axis shake (0-1)
